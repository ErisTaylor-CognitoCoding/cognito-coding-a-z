# Cognito Coding — Learn to Code A–Z

> **Learn To Code By Building Real Things.**  
> No fluff. No theory dumps. Just real code.

This is the companion repo to the **[Cognito Coding YouTube channel](https://www.youtube.com/@CognitoCoding01)**.

Every coding concept that gets its own Short gets its own folder here — with the analogy, the explanation, and something you can try right now.

---

## How to use this repo

1. Pick a concept from the list below.
2. Open the folder — read the `README.md` for the explanation.
3. **Python concepts:** open `python/example.py` — run it, break it, change it.
4. **Scratch concepts:** click the Scratch project link and remix it.
5. Watch the matching Short on YouTube.

No experience needed. Start anywhere.

## The three series

### 🐍 Python — One Concept, One Board (Mon + Thu Shorts)
Pure Python concepts, jargon-free. Each one maps to a Short where the concept gets drawn on a whiteboard, then coded in a single file. Start here if you've never coded before.

### 🌾 Farmer Was Replaced — Saturday Shorts
Python concepts taught through [The Farmer Was Replaced](https://store.steampowered.com/app/2060160/The_Farmer_Was_Replaced/) — a game where you write real Python to program a farming drone. Every concept shows up in the game first, then in plain code you can run anywhere.

### 🎮 Scratch — Wednesday Shorts
Block coding concepts from the Scratch series. Scratch code lives on [scratch.mit.edu](https://scratch.mit.edu) — every project is public and remixable. The folders here explain the *why* behind the blocks and link to the live project. See [scratch/README.md](./scratch/) for the full Scratch index.

## Running the Python examples

All Python examples are plain Python 3 — no libraries or installs needed.

```bash
python variables/python/example.py
```

Change the value, add a print, break something — that's how you learn.
