# Cognito Coding — Learn to Code A–Z

> **Learn To Code By Building Real Things.**  
> No fluff. No theory dumps. Just real code.

This is the companion repo to the **[Cognito Coding YouTube channel](https://www.youtube.com/@CognitoCoding01)**.

Every coding concept that gets its own Short on the channel gets its own folder here — with the analogy, the classroom explanation, and a working example you can run right now.

---

## How to use this repo

1. Pick a concept from the list below.
2. Open the folder — read the `README.md` for the explanation.
3. Open `python/example.py` — run it, break it, change it.
4. Watch the matching Short on YouTube if you want to see it explained on a whiteboard.

No experience needed. Start at the top, work down.

## Running the examples

All examples are plain Python 3 — no libraries needed.

```bash
python variables/python/example.py
```

Change the value, add a print, break something — that's how you learn.
