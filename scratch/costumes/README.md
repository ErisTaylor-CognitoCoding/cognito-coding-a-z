# Costumes — Animate with Looks

> **Switch between images rapidly to create the illusion of movement.**

---

## The analogy

A flipbook. Draw a slightly different image on each page, then flick through them fast — and it looks like the character is walking.

Scratch sprites have "costumes" — different images of the same character in different poses. Switch between them quickly and the sprite appears to move. That's animation.

---

## The blocks

```scratchblocks
when green flag clicked
forever
    next costume
    wait (0.1) seconds
end
```

Every 0.1 seconds, switch to the next costume. Loop forever. The sprite appears to animate.

---

## Why it matters

Animation in any medium is just fast image-switching. CSS animations, sprite sheets in game engines, GIF files — all the same principle. In Scratch you *see* it happen at a speed you control. Understanding this makes spritesheet animation in Unity or Pygame make immediate sense.
