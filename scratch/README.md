# Scratch — Block Coding Concepts

> **The same ideas as Python, but built with coloured blocks instead of typed text.**

This section is the companion to the **Wednesday Scratch Concept Shorts** on [@CognitoCoding01](https://www.youtube.com/@CognitoCoding01).

Each folder = one Scratch concept. One Short. One plain explanation. A link to the live Scratch project so you can remix it yourself.

---

## Why Scratch?

Scratch removes the "syntax wall" — you can't make a typo because you snap blocks together like Lego. That means you can focus on the idea (loops, decisions, events) without worrying about colons and brackets.

Every concept here transfers directly to Python, JavaScript, or any other language. Learn it in Scratch first, then spot it again when you meet it in text code.
