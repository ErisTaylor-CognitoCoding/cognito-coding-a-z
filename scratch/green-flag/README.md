# Green Flag — The Start Block

> **The block that tells Scratch: "when I click play, begin here."**

---

## The analogy

Every program needs a starting point. In Python, the code runs top-to-bottom from line 1. In Scratch, nothing happens until you click the green flag — and the "When green flag clicked" block is the instruction that says "start here."

It's the on-switch. No green flag block = your code never runs.

---

## The blocks

```scratchblocks
when green flag clicked
say [Hello!] for (2) seconds
```

One event block. One action block. That's a complete Scratch program.

---

## Why it matters

Every language has a starting point. Python runs from the top. JavaScript runs when the page loads or a button is clicked. Scratch runs when you click the flag. The concept is identical: something triggers the start, then the code runs.
